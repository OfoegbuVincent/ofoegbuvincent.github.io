Honeypot Setup & Analysis

Overview

This project involves setting up a Cowrie-based honeypot to capture unauthorized login attempts and analyze attacker behavior. The data collected is visualized and stored using the ELK stack for further investigation.

Setup

The honeypot was deployed on a Linux VM with the following configuration:

  • OS: Ubuntu 22.04 LTS
  • Cowrie SSH & Telnet honeypot
  • ELK stack for log collection and visualization
  • Custom scripts for alerting and monitoring

Findings

Over the first 2 weeks, the honeypot captured multiple unauthorized login attempts. Key insights include:

  • Common usernames attempted: admin, root, test
  • Top attacking IPs and geolocation mapping
  • Patterns of automated brute-force attacks

Conclusion

This honeypot project demonstrates the value of proactive monitoring and analysis in cybersecurity. Future work includes integrating additional alerting tools and simulating more complex attack scenarios.
